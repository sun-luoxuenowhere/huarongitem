# 目录文件说明
### node_modules项目依赖包
### build项目构建的配置目录，包括打包方式等
### config项目运行的配置目录，包括端口号等
### src/assets 放置一些图片，如logo等
### src/components 目录里面放了一个组件文件
### dist 项目编译后的文件目录
### App.vue 项目入口文件
### router 文件跳转路由
### main.js 引入资源根js文件
### static 开发中写的静态文件
### store 开发过程中提供一个仓库(Vuex)储存数据的使用；

# 项目启动说明
	>电脑需要安装node，如果没有node_modules需要打开控制台 cmd npm install进行安装；
	>启动项目npm run dev
	>访问localhost：8080
